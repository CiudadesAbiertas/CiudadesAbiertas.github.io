Consejos para visualizaciones de datos

Se han llevado a cabo dos actuaciones relacionadas con la visualización de datos. En la primera de ellas se analiza la capacidad de generar visualizaciones mediante sistemas externos (tipo Plotly) a partir de conjuntos de datos publicados en portales de datos abiertos. Una vez analizadas las herramientas disponibles se expone cómo se debería llevar a cabo su utilización desde un portal de datos abiertos para aplicar las visualizaciones a los conjuntos de datos que contenga. Se encuentra disponible para su descarga el Informe con el Análisis de Sistemas Externos de Visualización. Adicionalmente se ha desarrollado una extensión para CKAN denominada “Abrir Con”, publicada en la cuenta de GitHub de Ciudades Abiertas, que permite la integración sencilla con este tipo de herramientas.

La segunda actuación se ha centrado en el análisis de extensiones de visualización de recursos de CKAN. Estas extensiones se han analizado tanto desde el punto de vista técnico como funcional, contemplando en el análisis los siguientes tipos de visualización: tablas, mapas (visualización de información georreferenciada), gráficas y dashboard (tablero de visualizaciones). En el siguiente enlace se puede descargar el Informe con el Análisis de las Extensiones de Visualización para CKAN.

Se pueden consultar más detalles sobre los resultados obtenidos en la página sobre la actuación correspondiente al estudio de visualizaciones.
